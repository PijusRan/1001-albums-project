## 💡 Overview

This is a website with the purpose of browsing the famous list. It was created because there are no good variations of the list database.

## ✨ Features

- **📄 Generation:** Generates a random song from the list.
- **🔍 Listening history:** Let's you see what songs you listened to and how you liked them.
- **🔐 Local data management:** The ratings are safely stored in local memory, no database.
- **📱 Responsive Design:** Access on any device with adaptive design.
